1. Structure of the code:
1 main py file + 4 algorithm py files

2. Run the code:
1) Set directory to folder code
2) Add input data folder named as DATA to the code folder
3) Open terminal at the folder
4) Type python main.py -inst <filename> -alg [BnB|Approx|LS1|LS2] -time <cutoff in seconds> -seed <random seed>. Eg. python main.py -inst jazz -alg BnB -time 60 -seed 6140
5) Outputs are in the folder output
